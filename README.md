## 🌍 What is Hacktoberfest?

**Hacktoberfest 2025** is a month-long celebration of open source software, powered by **DigitalOcean**, **GitHub**, and the global developer community.  
Throughout October, contributors from around the world come together to:
- ✨ Learn Git & GitHub  
- 🧑‍💻 Contribute to open source projects  
- 🌱 Grow as developers  
- 🏆 Earn cool rewards and recognition  

---

## 🚀 How to Participate

1. **Register** on [hacktoberfest.com](https://hacktoberfest.com) with your GitHub account.  
2. **Fork** this repository to your own GitHub account.  
3. **Create a new branch** for your changes.  
4. **Add your contribution** — it can be a new feature, bug fix, documentation improvement, or creative idea.  
5. **Submit a Pull Request (PR)** and wait for it to be reviewed and merged.  

> 🏅 Submit **4 valid PRs** during October to complete Hacktoberfest and earn official rewards!
